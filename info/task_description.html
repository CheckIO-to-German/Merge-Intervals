You have a collection of intervals. You should merge overloped intervals.

Input: A list of tuples. Each tuple has two elements, both are int. The elements show beginning and ending of interval.

Output: A list of tuples, that has the same structure as input value.

Example: 

    merge_intervals([(1, 4), (2, 6), (8, 10), (12, 19)]) == [(1, 6), (8, 10), (12, 19)]
    merge_intervals([(1, 12), (2, 3), (4, 7)]) == [(1, 12)]

Precondition: intervals are sorted inside the list by first element; the first element of interval is always bigger than the second one.
